# Rey Lester Jariel — Portfolio Site

Personal portfolio for **Rey Lester Jariel**, Senior WordPress Engineer & AI Solutions Architect based in the Philippines.

## Live Site

Built with React 18 (UMD, no build step), plain CSS, and deployed via GitHub.

## Stack

- **React 18** — component-driven UI rendered client-side via UMD scripts
- **Archivo + Anton + JetBrains Mono** — type system via Google Fonts
- **Vanilla CSS** — custom design tokens, no framework
- **GitHub Pages** — deployment target

## Sections

- **Hero** — name, role, and primary CTA
- **About** — background, location, and project count stats
- **Services** — WordPress Development, WooCommerce, Performance & SEO, AI Integrations, Automation (n8n), REST APIs & Headless
- **Experience** — work history across senior WordPress and lead developer roles
- **Stack** — technologies and tools
- **Contact** — email CTA at jarielweb@gmail.com
